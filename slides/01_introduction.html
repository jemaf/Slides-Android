Certo... mas qual é a melhor plataforma?

Google Android

Sistema de código aberto voltado para plataforma móvel

Kernel baseado no Linux

Características principais:

  • Suporte ao multitasking
  • Grande flexibilidade
  • Comunicação entre processos por meio de mensagens

Google Android

IDE: Android Studio

Lnguagem: Java & Kotlin

Taxa de Registro: U$ 25,00 no momento da publicação do App.

Apple iOS

Sistema Operacional voltado para dispositivos móveis da Apple

Primeiro S.O. móvel de sucesso, lançado em iOS

Características principais:

  • Ecossistema Fechado
  • Alto rigor na avaliação dos aplicativos
  • Sem comunicação inter-processo

Apple iOS

IDE: XCode

Lnguagem: Objective-C e Swift

Taxa de Registro: U$ 99,00 a.a para publicar.

Windows Phone

Sistema operacional móvel lançado pela Microsoft

Primeira versão lançada em 2012

Características principais:

  • UX inovadora
  • Multitasking e comunicação inter-processo
  • Otimizações de consumo de energia

Descontinuado em 2017

Windows Phone

IDE: Visual Studio

Lnguagem: .NET

Taxa de Registro: de U$ 19,00 a U$ 99,00.
