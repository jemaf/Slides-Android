O Ecosssistema Android

Sistema Operacional mantido pelo Google*

Líder no seguimento de dispositivos móveis

Disponível para diversas plataformas

  • TVs
  • Relógios
  • Carros
  • Smartphones (é claro!!)

Fonte: Statista

A Open Handset Alliance

Consórcio formado por 84 empresas

Objetivo: Contribuir para criação e manutenção de um S.O móvel aberto

  • Fabricantes: HTC, Samsung, LG, Motorola, etc.
  • Operadoras: Nextel, T-Mobile, Telefonica, etc.
  • Semicondutores: Intel, Broadcom, NVidia, Qualcomm, etc.
  • Software: Google, eBay, Ascender, etc.
  • Comércio: Aplix, Noser, TAT, Wind River, etc.

Histórico de Versões

Última Versão: 8.0 (Oreo)

Principais Características

  • Kernel baseado em Linux
  • Linguagem de desenvolvimento: Java e Kotlin*
  • Open source, licensiado sob licença Apache Software Foundation
  • Sandboxing
  • Programação Intencional
  • Multitasking nativo
  • Cada aplicativo é executado como um processo do S.O.

Arquitetura

Android Runtime (ART)

Máquina Virtual criada com objetivo de substituir a Java Virtual Machine

Contém melhorias voltadas especificamente para execução em dispositivos móveis:

  • Melhor Consumo de Energia e Memória
  • Melhor Garbage Collector
  • Compilação Ahead-of-time (AOT)

Importante! Seu funcionamento é transparente para desenvolvedores e usuários
